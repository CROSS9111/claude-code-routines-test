Claude Code 向け無料セキュリティガイダンスプラグインをリリース ─ SQLi・XSS・シークレットハードコードなど25クラスを3層でリアルタイム検出
New Plugin Source: CyberSecurityNews / Help Net Security — 2026.05.27

Anthropic は5月27日、Claude Code の全プラン・全ユーザーに向けて無料のセキュリティガイダンスプラグインをリリースした。/plugins コマンドからプラグインマーケットプレイスに直接インストールでき、コードを書きながらリアルタイムで脆弱性パターンを検出する。

  • 検出対象（約25クラス）: SQL インジェクション、コマンドインジェクション、XSS、ハードコードされた API キーとシークレット、セキュアでないデシリアライゼーション、不適切な入力バリデーションなど
  • 3層レビュー体制: ①ファイル編集時、②モデルターン後、③コミット前、の3段階でコードをチェック。問題が検出されると、同一セッション内でインラインフィックスを提案する
  • 実績: 内部テストで PR のセキュリティ関連コメントを30〜40%削減。リリース24時間で157,000ダウンロードを記録
  • Claude Sandbox との併用: 同日リリースの Claude Sandbox と組み合わせることで、コーディングから実行まで一貫したセキュリティカバレッジが得られる
Dynamic Workflows 発表翌日の反響 ─ Bun の75万行 Rust 移植が「本当に動いた」ことで業界に衝撃、HN では制御機構への議論も
Reaction Source: Hacker News / TechCrunch — 2026.05.28-29

昨日（5/28）発表された Dynamic Workflows は、Bun の創設者 Jarred Sumner による Zig→Rust 全面移植（約75万行・11日間・既存テスト99.8%パス）を旗艦事例として大きな反響を呼んでいる。

  • Hacker News の主要議論: 「長時間実行セッションへの割り込み手段が不足している」という制御機構への要望が多い
  • Jarred Sumner 本人コメント: 「11日というのはフルタイムで使った場合。最初の3日は Workflow の設計と CLAUDE.md の整備に費やした」と補足
  • 利用要件: Max・Team・API プランはデフォルト有効。Free プランは対象外
MIT Technology Review「Code with Claude 2026 は開発の未来を見せた ─ 好むと好まざるとにかかわらず」
Analysis Source: MIT Technology Review — 2026.05.21

MIT Technology Review は5月21日付けで、Anthropic の開発者カンファレンス「Code with Claude 2026」を取り上げ、「AIコーディングの未来を見せた ─ 好むと好まざるとにかかわらず」と評した。

  • 三層構造の提示: ①ツール層（Claude Code CLI・MCP・Hooks）、②オーケストレーション層（Dynamic Workflows・Agent View）、③監督層（人間が担う設計・判断・倫理）
  • 「ツールを使うかどうかの選択権は人間にあるが、使わない選択のコストは年々高まっている」と締めくくった
