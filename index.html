Claude Code 週間制限を50%増量 ― 7月13日までの期間限定、Codex 対抗の明確な戦略転換
Limits Source: Pasquale Pillitteri / Releasebot — 2026.05.13-14

Anthropic が Pro・Max・Team・Enterprise（席ベース）全プランで Claude Code の週間使用上限を50%増量した。期間は2026年7月13日まで。この施策は5月6日の5時間ウィンドウ上限2倍化に続くもので、5週間に3度目の容量拡張となる。

  • 背景: Reddit で「Claude Code は品質は高いが上限がすぐ尽きて日常ドライバーにならない」「Codex の方が実際に使い続けられる」という声が急増していた
  • この増量でレート制限は Codex と同水準になったとする分析が複数の技術ブログで公開
  • 無料プランは対象外。期間終了後の恒久化については明言されていない
Anthropic、Agent SDK クレジット分離を6月15日より導入 ― サードパーティ利用は別枠へ
Billing Source: DevToolPicks / Gigazine — 2026.05.14

6月15日以降、Claude 有料プランのサブスクリプション利用枠が「ファーストパーティ（Claude.ai・Claude Code CLI）」と「Agent SDK・サードパーティアプリ」で別々のクレジットプールに分離される。

  • Pro: 月$20相当 / Max 5x: 月$100相当 / Max 20x: 月$200相当のAgent SDKクレジットを付与
  • OpenClaw など サードパーティ AIエージェントハーネスのアクセスが再び有効化される
  • Zed エディタなど IDE 統合への影響も明記 ― 既存ユーザーは6月15日前に設定移行が必要
  • クレジットの受け取りは Anthropic からのアカウントメールに記載の手順で申請

「Claude Code が奇妙になった」― power users の反発とサブスクリプション混乱の経緯
Analysis Source: Anthony Maio / Substack / Latent Space — 2026.05

過去6週間で3回のレート制限変更・Agent SDK分離・スマートルーティングの導入が重なり、上級ユーザーの間で「どのプランが何トークン使えるか分からなくなった」という混乱が広がった。Simon Willison は「複数の投稿を読み、エイリアスを切り替え、隠れた設定を推測しなければならないなら、Claude Code はもっとシンプルになる必要がある」と指摘。Anthropic は50%増量でその批判を部分的に吸収したが、6月15日の課金分離が新たな理解コストを生むとする懸念も残る。
