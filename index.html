Claude Code Auto Mode ― 承認ゲート付き自律コーディングが一般提供へ
New Feature Source: Anthropic / InfoQ — 2026.05

Anthropicが「Auto Mode」を一般提供。開発者が目標を定義すると、Claude Codeがコード生成・実行・ツール呼び出し・反復改善を自律処理し、機密性の高い操作のみ人間の承認を要求する「ヒューマン・アプルーバル・ゲート」方式を採用。

  • 高速フィルターで大多数のツール呼び出しを自動判定、リスクが不確かな操作のみ深層分析にエスカレーション
  • 承認確認時はスピナーが赤色に変わり視覚的にわかりやすく通知
  • Auto Mode ハード拒否ルールにより特定アクションを無条件ブロック可能
