Microsoft が Claude Code ライセンスを 6月30日で全廃 ─ Experiences+Devices 部門の数千人を GitHub Copilot CLI へ強制移行
BreakingSource: CyberNews / OpenTools.ai — 2026.06.01

Microsoft は Experiences+Devices 部門において、6月30日を期限に Claude Code ライセンスを廃止し、GitHub Copilot CLI への移行を全エンジニアに義務付けた。

  • 背景: Uber が2026年の AI 予算全額をわずか4ヶ月で Claude Code のトークン費用に使い切った事例が広く報道されており、Microsoft でも同様の構図が起きていたとみられる
  • ROI の失敗: AI Weekly は「企業向け AI ROI の失敗として、Claude Code の撤退は象徴的な事例になる可能性がある」と分析
  • エンジニアの反応: 社内からは「ツールの品質よりコスト管理を優先した判断」という批判的な声も漏れ伝わっている
Anthropic、Opus 4.8 を全プランのデフォルトに ─ Dynamic Workflows・セキュリティプラグイン・MCP OAuth が GA
UpdateSource: MindStudio / Claude Platform Docs — 2026.06.01

Anthropic は Claude Opus 4.8 を Max・Team Premium・Enterprise pay-as-you-go・Anthropic API アカウントの新デフォルトモデルとした。Dynamic Workflows・セキュリティプラグイン・MCP OAuth・Fast Mode などが一斉に GA となった。

  • Dynamic Workflows: エージェントが実行中にワークフローを動的に調整。スクリプト1本で多数のサブエージェントをオーケストレーション可能に
  • MCP OAuth サポート: OAuth 2.0 フローが使えるようになり、企業の既存 SSO システムとの統合が容易に
  • Fast Mode: Opus 4.8 を使いつつ出力速度を大幅向上。/fast コマンドで切り替え可能
