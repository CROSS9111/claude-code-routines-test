Claude Managed Agents 詳細解説 ─ Dreaming・Outcomes・MCP Tunnels・セルフホストサンドボックスの全体像が明らかに
Platform Source: 9to5Mac / The Decoder / InfoQ — 2026.05.19〜22

Code with Claude カンファレンス終了を受け、複数のテックメディアが Claude Managed Agents の新機能群を改めて詳細解説。各機能の用途と制限が整理された。

  • Dreaming（夢）: エージェントがタスク終了後にトランスクリプトを内省し「次回の自分への注記」を生成。同じコードベースで後続エージェントがこの注記を参照し、過去のエラーパターンを回避できる。複数タスクを横断してパターンを統合・圧縮する機能も含む
  • Outcomes（成果基準）: 「テストが全て通ること」「PR レビューが承認されること」など成功条件を事前定義し、エージェントがその条件達成を目標に自律的に作業する仕組み
  • セルフホストサンドボックス（Public Beta）: Cloudflare・Daytona・Modal・Vercel をパートナーとして採用。エージェントのオーケストレーション・コンテキスト管理は Anthropic インフラ、ツール実行は自社環境で行う分割アーキテクチャ
  • MCP Tunnels（Limited Research Preview）: 社内 MCP サーバーへのプライベートルーティング。VPN 不要・ファイアウォール変更不要で社内ナレッジベースをリアルタイム参照可能。現時点はアクセスリクエスト制
