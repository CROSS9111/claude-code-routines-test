KPMG × Anthropic 戦略的グローバル同盟 ─ Digital Gateway Powered by Claude で27万6千人が Claude を活用
Enterprise Source: Anthropic / KPMG — 2026.05.19

本日、KPMG と Anthropic が世界的な戦略同盟を締結し、「KPMG Digital Gateway Powered by Claude」を正式発表した。KPMG の276,000人超の全従業員が Claude へのアクセスを得る。

  • Digital Gateway は Microsoft Azure 上に構築された KPMG のグローバル技術プラットフォームで、税務インサイト・独自ツール・クライアントデータを一元管理している
  • AI エージェントを使って税規制変更への対応ワークフローを構築するのに、以前は数週間かかっていた作業が「数分」に短縮されたと説明
  • プライベートエクイティ向け製品にも Claude を組み込み、ポートフォリオ企業向けの新製品を共同開発予定
  • 大手会計事務所の Claude 採用は PwC に続いて2社目。Big Four のうち半数が Anthropic と提携したことになる
Claude Code 最新アップデート ─ /resume でバックグラウンドセッション対応、起動ハング75秒を修正
Update Source: Claude Code Changelog / Releasebot — 2026.05.19

最新 Claude Code リリースはバックグラウンドセッション対応と信頼性向上が中心。

  • /resume のバックグラウンドセッション対応 ─ claude --bg や Agent View で開始したセッションが一覧表示されるようになった
  • バックグラウンドサブエージェント完了通知に経過時間を追加（例: 「Agent completed · 3h 2m 5s」）
  • /model は現セッションのみモデルを変更するよう変更。新規セッションのデフォルトはモデルピッカーの「d」キーで設定
  • 起動時に最大75秒ハングしていた問題を修正 ─ サイドチャネルのAPI呼び出しが15秒でタイムアウトするよう改善
