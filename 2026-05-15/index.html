Microsoft、社内 Claude Code ライセンスをキャンセル ― 6月30日までに GitHub Copilot CLI へ全面移行
Breaking Source: Windows Central / TechRadar — 2026.05.15

Microsoft の Experiences + Devices 部門が、社員に付与していた Claude Code ライセンスの大半をキャンセルし、GitHub Copilot CLI への移行を命じた。移行期限は2026年6月30日（Microsoft 会計年度末）。

  • 2025年12月に数千人に展開した Claude Code が想定外に普及し、Copilot CLI の社内採用を妨げていた
  • 財務的動機も指摘: Claude Code は Anthropic への外部コストが発生するが、Copilot CLI は Microsoft が直接収益を得るモデル
  • Anthropic との「Foundry 契約」は維持。Copilot CLI 経由で Claude モデル自体は引き続き利用可能
  • 社内エンジニアの反発は大きく「品質が落ちる」「生産性が下がる」という声が上がっている
Claude Code 創始者、「毎晩数千のサブエージェントをスマートフォンから管理」― Sequoia CA との対談で衝撃の作業スタイルを披露
Insight Source: Let's Data Science — 2026.05.15

Claude Code のリードエンジニア Boris Cherny 氏が Sequoia Capital とのインタビューで衝撃的な作業スタイルを披露。通常「5〜10のセッション」を並行維持し、各セッションが数十のサブエージェントをスポーン。鍵となる機能は /loops と Routines（サーバーサイドの定期タスク）で、「Claude Code はコーディングツールではなく、エージェント実行インフラだ」という認識が注目された。

Anthropic、サードパーティエージェントを Claude サブスクリプションで再有効化 ― 6月15日からクレジット分離方式で
Billing Source: VentureBeat / InfoWorld — 2026.05.15

Anthropic が6月15日から導入するクレジット分離の全詳細を公式発表。Pro: 月$20 / Max 5x: 月$100 / Max 20x: 月$200 相当の Agent SDK クレジットを付与。Claude Code CLI・GitHub Actions は引き続き通常のプラン枠から利用可能。
